Catalog bulk import

Paste a CSV (or choose a file) exported from your spreadsheet. Each row upserts a book by its slug (auto-built from title + author + year), so re-importing updates existing books — including their tags — instead of duplicating. New rows import as drafts unless the CSV sets status.

1 · Get the template

Download catalog-template.csv, open it in Google Sheets / Excel, fill rows, then export as CSV.

Multi-value fields (tropes, mood, vibes, themes, content_warnings, representation, setting, kink_tags, mc_archetype, li_archetype, comp_titles, heat_type) take several values separated by a semicolon, e.g. enemies-to-lovers; fae; slow-burn — these become catalog tags. Spice: spice_level 0–5, door = open/fade/closed, spice_frequency = none/rare/occasional/frequent/constant. Booleans accept true/false. Leave a cell blank to skip it.
